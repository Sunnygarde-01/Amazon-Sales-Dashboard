#📦 **Amazon Sales Performance Analytics**

Overview :-
Transforming raw e-commerce data into actionable business intelligence. This project features a comprehensive Power BI Dashboard designed to track, analyze, and optimize Amazon sales performance. It provides a deep dive into revenue trends, order fulfillment, and consumer demographics.

❓ The Challenge (Problem Statement)
In the fast-paced world of e-commerce, raw data is overwhelming. The primary challenge was to convert a massive dataset of 128,000+ transactions into meaningful insights.

• **Manual Data Fatigue**: Difficulty in tracking monthly growth and revenue shifts manually.

• **Operational Blindspots**: No clear visibility on order cancellation rates or fulfillment efficiency (Amazon vs. Merchant).

• **Targeting Issues**: Lack of data-driven evidence on which product categories or sizes dominate specific geographical regions.

💡 The Innovation (Solution)
I developed a dynamic, end-to-end analytical solution that automates the insight-generation process:

• **Interactive Visualization**: Built a multi-page Power BI dashboard that simplifies complex sales metrics.

• **KPI Optimization**: Integrated high-level cards for Total Revenue (₹78.5M), Total Orders, and Success Rates.

• ** Advanced Filtering**: Implemented slicers for Date, Category, and Fulfillment Method to allow stakeholders to perform "drill-down" analysis.

• **Data Sanitization**: Performed rigorous data cleaning in Excel and Python to ensure accuracy in reporting.

🛠️ Tech Stack
• **Visualization**: Power BI Desktop

• **Data Processing**: Microsoft Excel (Advanced)

• **Language**: Python (for data profiling)

• **Environment**: GitHub for Version Control

📊 Key Insights Extracted
• **Peak Performance**: Identified April as the highest revenue-generating month (~₹28.7M).

• **Product Dominance**: T-shirts and Shirts emerged as the top-selling categories.

• **Regional Hubs**: Maharashtra and Karnataka were identified as the primary shipping destinations.

• **Fulfillment**: Visualized the ratio between "Easy Ship" and "Amazon" logistics to evaluate delivery speeds.

🚀 How to Use

Explore the interactive charts to view specific state or product performance.
